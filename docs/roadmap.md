# Roadmap

## v0.1.0 — Foundation (Current)

- [x] FastAPI backend with health/version endpoints
- [x] Tax computation engine (old & new regime)
- [x] SQLAlchemy async database layer
- [x] PDF and CSV parsers
- [x] Zerodha broker import
- [x] Excel report generation
- [x] React frontend dashboard
- [x] Docker & Docker Compose
- [x] CI/CD with GitHub Actions
- [x] Pre-commit hooks and code quality tooling

## v0.2.0 — Document Processing

- [ ] Form 16 PDF parser with field extraction
- [ ] AIS (Annual Information Statement) import
- [ ] Form 26AS import and reconciliation
- [ ] Capital gains computation (equity, debt, crypto)
- [ ] Multi-broker support (Groww, Upstox, ICICI Direct)

## v0.3.0 — ITR Filing

- [ ] ITR-1, ITR-2, ITR-3 form generation
- [ ] JSON export for e-filing portal
- [ ] Deduction optimizer (regime comparison)
- [ ] Advance tax calculator
- [ ] TDS reconciliation

## v0.4.0 — AI Copilot

- [ ] Natural language tax queries
- [ ] Document understanding with LLM
- [ ] Tax planning recommendations
- [ ] Anomaly detection in broker statements
- [ ] Conversational filing assistant

## v0.5.0 — Enterprise
